SISTEMA DE NUMERAÇÃO

Um sistema de numeração é a organização e representação dos algarismos.
O nosso sistema de numeração é o decimal e possui base 10, ou seja, 10 dígitos de representação (0 a 9) para os números. Já o sistema de numeração binário possui base 2 e usa apenas 2 dígitos: 0 e 1.
Os dois sistemas são considerados posicionais, onde cada dígito tem valor posicional.
Exemplo: 104 e 345.
Os dois números possuem o dígito 4, porém no primeiro possui o valor de 4 unidades, já no segundo, possui o valor de 4 dezenas ou 40.

CONVERSÃO BINÁRIO <-> DECIMAL

Para converter um número binário para base 10(decimal), basta decompor o número binário.
Exemplo: 10100 na base 2.
0 × 2^0 + 0 × 2^1 + 1 × 2^2 + 0 × 2^3 + 1 × 2^4 = 2^2 + 2^4 = 4 + 16 = 20
Lembrando que o valor do expoente equivale a posição de cada dígito começando da direita para a esquerda, a partir do 0.

Para converter um número decimal para base 2, utilizaremos o método de divisões sucessivas, que consiste em dividir o número a ser convertido por 2 de maneira a obter o resto que equivale a posição 0(posição mais a direita) e repetir o processo.
Exemplo: 25.
Posição 0 -> Divida 25 por 2. Resultado é 12 e o resto é 1.
Posição 1 -> Divida 12 por 2. Resultado é 6 e o resto é 0.
Posição 2 -> Divida 6 por 2. Resultado é 3 e o resto é 0.
Posição 3 -> Divida 3 por 2. Resultado é 1 e o resto é 1.
Posição 4 -> Divida 1 por 2. Resultado é 0 e o resto é 1.
Assim, 25 em decimal é 11001 em binário.

RELAÇÃO NÚMEROS BINÁRIOS E COMPUTADOR

Para entender a relação entre os binários e o computador, é importante saber que cada dígito de um número binário(0 ou 1) é chamado de bit.
Computadores armazenam e recuperam dados em forma de bits. O conjunto de 8 bits é chamado de byte, que pode representar até 256 valores diferentes.
O computador armazena esses bytes na memória principal e esses dados armazenados são interpretados pelo processador (ou CPU - central processing unit) que também executa as operações lógicas e aritméticas.
Entender essa relação é crucial para qualquer pessoa envolvida no meio tecnológico. Tendo como objetivo entender como a informação é processada e armazenada nos níveis básicos do sistema computacional.

TABELA ASCII (American Standard Code for Information Interchange)

Toda informação para ser armazenada em um computador precisa de uma representação binária.
A tabela ASCII é um conjunto de códigos numéricos que atribui um único número para cada caractere (letras maiúsculas e minúsculas, números, pontuações e caracteres especiais).
Um código ASCII é a representação numérica de um caractere.
Exemplo: O código ASCII para a letra 'A' é 01000001 em binário.
Essa tabela foi inicialmente criada apenas com a representação dos números, as 26 letras do alfabeto inglês e alguns caracteres especiais e de controle, porém, para suprir a necessidade de incluir caracteres de outros alfabetos, foi futuramente expandida para 256 caracteres, todos com 8 bits.
Cabe dizer que o padrão ASCII ainda não atendia idiomas, como árabe e chinês, e por isso, houve a criação de novos padrões, sendo o mais universal, a representação Unicode.

CONVERSÃO TEXTO <-> BINÁRIO

Para realizar a conversão de texto para binário basta seguir alguns passos.
1. Obter o código ASCII de cada caracter do texto.
2. Converter cada código ASCII para binário.
3. Combinar as representações binárias.

Exemplo: Convertendo a palavra "TEXTO".
1. Obter os códigos na tabela ASCII:
T:84 E:69 X:88 O:79
2. Converter para binário utilizando o método ensinado.
T:01010100 E:01000101 X:01011000 T:01010100 O:01001111
3. Unir os binários:
01010100 01000101 01011000 01010100 01001111

Para converter de Binário para texto, basta seguir os passos inversos:
1. Dividir a sequência binário em blocos de 8 bits.
2. Converter cada byte para decimal.
3. Obter cada caractere correspondete ao código ASCII
4. Unir os caracteres e formar o texto.
